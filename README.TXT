Para ejecutar el programa, asegurarse de haber descomprimido el archivo .tar.gz

Estando en la altura de los archivos, escribir "make" en la terminal, y luego "./Battleship.exe", y seguir
las instrucciones dadas en pantalla

Si se quiere limpiar los archivos creados por el Makefile, escribir "make clean"
Si se quiere analizar el programa en buscas de leaks de memoria, escribir "make valgrind"
